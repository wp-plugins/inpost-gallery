== Description ==

Powerfull and very pleasant photo gallery plugin to work with images in WordPress. 
The Backend is powered with visual shortcodes management where you only by mouse can change view of your post gallery on front. 
You do not need go to another pages with amount of options. Just install and use, 
all you need it is popup with shortcode options on your current.

Good for photographers. Supports: multiple images uploading, css effects (like round corners of images with shadow).

http://www.youtube.com/watch?v=FA0J-bdovXU

== Installation ==
* Download to your plugin directory or simply install via Wordpress admin interface.
* Activate.
* Use.

== Changelog ==

= 2.0.0 =
Rewritten code. Added visual shortcode interface in backend, more flexible options for shortcode.

= 1.1.5 =
In <your_site>/wp-admin/options-general.php?page=inpost-gallery-settings new ability to set custom post types (post and page too) where you want to use plugin.

= 1.1.4 =
Added attribute in shortcode "border". Example: [inpost_gallery post_id=327 random=-1 thumb_width=150 thumb_height=150 border="solid 2px #000"]. Useful to set border around of image by CCS syntax.

= 1.1.3 =
2 big improvements: multiple image adding and categorizing by groups. Them both very convinient for photographers.

= 1.1.2 =
